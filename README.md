# Name-Gender Classification API

## 📌 Overview
This is a RESTful API built with Express.js that predicts the gender of a person based on their name. The application takes a name as input and returns the likely gender using an external API.

## 🚀 Features
- Accepts a name as input via query parameters
- Returns predicted gender
- Simple and fast API response
- Built with Node.js and Express

## 🛠️ Technologies Used
- Node.js
- Express.js
- Fetch (for API requests)
